```batch
  cd LOCATION
```

## Features

- Custom jump interval
- Debug features
- Auto eat
- Advanced anti afk
- Auto selling on ecconomy servers
- Custom hit delay (works on all versions)
- Hold mode (no toggle mode yet)
- Spam mode which will spam the chat with a custom watermark
